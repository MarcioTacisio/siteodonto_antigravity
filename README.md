# OralCure Premium | Landing Page Odontológica de Alta Performance

Uma Landing Page ultra moderna, sofisticada e totalmente responsiva desenvolvida para a **OralCure Premium**, uma clínica odontológica fictícia de alto padrão. O design foi concebido com foco em conversão de leads, transmissão de autoridade, inovação tecnológica e absoluto acolhimento ao paciente.

---

## 🌟 Principais Recursos e Seções

1. **Navbar Fixa e Fluida**:
   - Transição inteligente ao rolar a página (redução de altura e ampliação de desfoque).
   - Menu hambúrguer mobile responsivo animado via JavaScript nativo.
2. **Hero Section**:
   - Headline impactante voltada para autoestima e tecnologia.
   - Chamadas de ação (CTAs) em destaque e integradas com WhatsApp.
   - Imagens integradas a formas geométricas com gradiente e efeitos de *blur* e *glassmorphism*.
3. **Cards de Serviços Interativos**:
   - 6 especialidades estruturadas em grid responsivo com efeitos de hover avançados e borda neon suave.
4. **Sobre a Clínica**:
   - Seção dividida (imagem institucional + dados de crescimento e diferenciais de SPA odontológico).
5. **Diferenciais Estruturados**:
   - 6 pilares de atendimento com ícones e microinterações intuitivas.
6. **Depoimentos de Pacientes**:
   - Avaliações em formato de cards premium com fotos de pacientes, estrelas douradas de satisfação e relatos reais de transformação.
7. **CTA Final e Rodapé**:
   - Chamada direta de alto impacto para agendamentos rápidos via WhatsApp.
   - Rodapé minimalista com dados de contato e links funcionais de redes sociais.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5 Semântico**: Estrutura robusta focada em SEO e acessibilidade.
- **Tailwind CSS (v3 via Play CDN)**: Estilização modular rápida, responsiva e moderna com paleta de cores personalizada e efeitos de brilho neon.
- **Lucide Icons**: Pacote de ícones vetoriais modernos e minimalistas de carregamento instantâneo.
- **Google Fonts**: Tipografias exclusivas ("Outfit" para títulos premium e "Inter" para textos legíveis).
- **Vanilla JS**: Microinterações nativas ultraleves (sem dependência de frameworks pesados).

## 🚀 Como Executar Localmente

Como a aplicação é estática e otimizada via CDN, você não precisa instalar nenhuma dependência de build:

1. Clone o repositório em sua máquina:
   ```bash
   git clone https://github.com/MarcioTacisio/siteodonto_antigravity.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd mariofono_antigravity
   ```
3. Dê um duplo clique no arquivo `index.html` ou use uma extensão de servidor local (como *Live Server* do VS Code) para abrir no seu navegador de preferência.

## 📦 Como Publicar no GitHub

Caso queira vincular e realizar o push do seu repositório local diretamente no GitHub, execute o arquivo script `publish.bat` (no Windows) ou utilize os seguintes comandos no terminal:

```bash
# Inicializar o repositório Git local
git init

# Adicionar todos os arquivos rastreáveis
git add .

# Realizar o commit inicial de configuração
git commit -m "chore: add gitignore, env.example and readme documentation"

# Fazer o commit da Landing Page principal
git commit -m "feat: add premium dental clinic landing page code"

# Configurar a branch padrão como 'main'
git branch -M main

# Conectar ao repositório remoto do GitHub
git remote add origin https://github.com/MarcioTacisio/siteodonto_antigravity.git

# Enviar os commits locais para o GitHub
git push -u origin main
```
